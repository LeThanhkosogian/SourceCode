  • flex: flex-grow | flex-shrink | flex-basis

11. Quy ước đạt tên class BEM

  1. BEM:
    • Là tiêu chuẩn đặt tên class khi viết CSS
  2. Ý nghĩa:
    • Viết tắt của: Block Element Modifier
    • Block: khối
    • Element: thành phần trong khối
    • Modifier: bố sung ý nghĩa cho 'Block' và 'Element'
  3. Tại sao phải sử sụng BEM:
    • Mỗi người đặt 1 kiểu
    • Members đặt class trùng nhau, CSS đè lên nhau
  4. Cú pháp
    • .block
    • .block__element
    • .block--modifier
    • .block__element--modifier
  5. Tính ứng dụng
    • Xây dựng layout website
    • Xây dựng thành phần trên website
  6. Ưu điểm
    • Tính rõ ràng
    • Tái sử dụng dễ dàng
    • Giúp cả team làm việc với nhau dễ dàng
    • Tính module, không lo CSS đè lên nhau
  7. Nhược điểm
    • Tên class dài
    • Một số người cho là xấu
  8. Khi nào dùng BEM thì phù hợp?
    • Dự án nhiều members
    • Dự án lớn, số lượng pages nhiều hoặc số lượng các thành phần trên giao diện nhiều
